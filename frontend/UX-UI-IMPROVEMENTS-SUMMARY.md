## 🎨 **Детали улучшений меню**

### **Новая карточка меню (EnhancedMenuCard)**

#### **Визуальные улучшения**
- **Высота изображения**: 220px для консистентности
- **Закругленные углы**: 16px для современного вида
- **Тени**: Многослойные тени для глубины
- **Hover эффекты**: Подъем карточки на 4px при наведении

#### **Интерактивные элементы**
- **Бейджи**: Категория (верхний правый) и цена (нижний левый)
- **Раскрывающиеся детали**: Плавная анимация показа/скрытия
- **Пищевая ценность**: Сетка с БЖУ и калориями
- **Кнопка "Подробнее"**: С анимированной стрелкой

#### **Адаптивная сетка**
```css
/* Десктоп (1200px+) */
grid-template-columns: repeat(auto-fill, minmax(320px, 1fr));

/* Планшет (768px-1200px) */
grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));

/* Мобильный (до 768px) */
grid-template-columns: 1fr;
max-width: 400px;
```

### **Улучшенная типографика**
- **Заголовки**: 1.25rem, жирность 700
- **Описания**: 0.875rem, цвет #6b7280
- **Цены**: 0.875rem, жирность 700, цвет #8B4513
- **Категории**: 0.75rem, жирность 600

### **Цветовая схема**
- **Основной**: #8B4513 (коричневый кофе)
- **Фон карточек**: #ffffff (белый)
- **Текст**: #1f2937 (темно-серый)
- **Вторичный текст**: #6b7280 (серый)
- **Бейджи**: #f8fafc (светло-серый фон)
